# Abdul Wahab Suhail — Portfolio

Personal portfolio site built with React 18, Vite, and Tailwind CSS. Ready for deployment on [Vercel](https://vercel.com).

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy to Vercel

1. Push this project to a GitHub repository.
2. Go to [vercel.com/new](https://vercel.com/new) and import the repo.
3. Vercel auto-detects Vite — click **Deploy**.

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Build

```bash
npm run build
npm run preview
```
